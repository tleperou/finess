Only native HTML elements ..

## Anchors

Anchors are links ..

```bash
<a href="#">Read article</a>
```

## Buttons

Button are actionable ..

```html
<button type="submit">Contact us</button>
```

## Details

Details are enclosure ..

### Grouped details

With grouped details, only one of them remain open.

```html
<details name="faq">
  <summary>Where to find us?</summary>
  <p>
    Ut ac sem rutrum, convallis nulla tincidunt, tempor dolor. Sed ac leo
    dictum, pulvinar urna quis, suscipit leo. Vivamus eu porta mauris, eget
    efficitur nisi
  </p>
</details>
<details name="faq">
  <summary>How to contact us?</summary>
  <p>
    Ut ac sem rutrum, convallis nulla tincidunt, tempor dolor. Sed ac leo
    dictum, pulvinar urna quis, suscipit leo. Vivamus eu porta mauris, eget
    efficitur nisi
  </p>
</details>
```

## Dialogs

Dialogs are container that ..

```html
<dialog>Hello</dialog>
```

## Separators

Separators are ..

```html
<hr />
```

## Inputs

```html
<input type="text" />
```

## Navigations

### Horisontal navigation

```html
<nav aria-orientation="horizontal">
  <ul>
    <li>Tab 1</li>
    <li>Tab 2</li>
    <li>Tab 3</li>
  </ul>
</nav>
```

### Vertical navigation

```html
<nav aria-orientation="vertical">
  <ul>
    <li>Link to 1</li>
    <li>Link to 2</li>
    <li>Link to 3</li>
  </ul>
</nav>
```
