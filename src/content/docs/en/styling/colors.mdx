Finess defines colors upon the [`<system-color>`](https://developer.mozilla.org/en-US/docs/Web/CSS/system-color#mark) of the device with the users' contrast preferences, and within p3 color space over RGB by using [`color-mix`](https://developer.mozilla.org/en-US/docs/Web/CSS/color_value/color-mix).

<aside>Personalise the color palettes using the dark color schema. Use a proper p3 color picker to empower the best of the colors.</aside>

### Usage

```css "var(--color-on-surface-1)" "var(--color-on-surface-2)" "var(--color-surface-1)"
hgroup {
  display: flex;
  flex-direction: column;
  row-gap: var(--row-gap-1);
  background-color: var(--color-surface-1);

  & h1 {
    color: var(--color-on-surface-1);
  }

  & small {
    color: var(--color-on-surface-2);
  }
}
```
