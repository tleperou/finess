Colors provide low and high contrast levels for users that turn them on. They also use the user's device preferences such as `Canvas` and `AccentColor` colors. They leverage high dynamic range color with the the p3 color space when supported.

### Usage

```css "var(--color-surface-1)" "var(--color-on-surface-1)" "var(--color-on-surface-2)"
hgroup {
  display: flex;
  flex-direction: column;
  row-gap: var(--row-gap-1);

  & h1 {
    color: var(--color-accent-1);
  }

  & small {
    color: var(--color-on-accent-2);
    background-color: var(--color-accent-1);
  }
}
```
